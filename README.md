## Tech Stack
- React






## Run Locally

Clone the project

```bash
  git clone https://github.com/SyedaMahamFahim/Leader-Board-React.git
```

Go to the project directory

```bash
  cd leader-board-react

```

Install dependencies

```bash
  npm install
```

Start the server

```bash
  npm start
```
